☁️ Deploy Backend to Render (Free)

1. Go to render.com → New Web Service → Connect your new repo
2. Build: pip install -r requirements.txt
3. Start: uvicorn backend.server:app --host 0.0.0.0 --port $PORT
4. Env vars: OPENAI_API_KEY, VOYAGE_API_KEY
5. Your API will be at https://your-app.onrender.com/docs

Open Render.com
